# 快捷键

说明：不同版本可能存在差异，请以当前版本的帮助面板为准。

## 已确认高频键位

| 快捷键 | 作用 | 场景 |
| --- | --- | --- |
| `Tab` | 切换模式 | 在 Build/Plan 等模式间切换 |
| `Enter` | 提交输入 | 发送当前指令 |

## 会话效率建议（比背快捷键更重要）

1. 用短指令分步推进，不要一次输入超长需求
2. 每次任务要求“先计划后执行”
3. 每步结束都要求输出“改动 + 验证 + 风险”

## 推荐肌肉记忆

1. `Tab` 选模式
2. 输入任务边界
3. `Enter` 提交
4. 检查结果后再进入下一步

## 如何查看你当前版本的完整键位

1. 启动 `opencode`
2. 执行 `/help`
3. 在交互面板查看快捷键说明
